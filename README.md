# SmartTV
> **Fighting brainrot with healthy media.** This is a curated collection of high-quality content for a more intentional viewing experience.

Contributions are welcome! Help us replace mindless scrolling with meaningful discovery.

---

### How to Sync
Stay updated using one of the two methods below:

#### 1. Automatic Sync (Recommended)
Keep your browser bookmarks updated in real-time using the **[Floccus extension](https://floccus.org/)**:
* **Connection:** Create a new **Git** sync profile pointing to this repo.
* **Strategy:** Choose **Merge** (to contribute) or **Download** (to just watch).
* **Mapping:** Select your local "SmartTV" folder to sync with `bookmarks.xbel` or `bookmarks.json`.

#### 2. Manual Import
1. Download the `bookmarks.html` file from this repository.
2. Open your browser's **Bookmark Manager** (`Ctrl+Shift+O` or `Cmd+Option+B`).
3. Click the **Menu** (three dots) and select **Import Bookmarks**.
4. Upload the file.

---

### How to Contribute
* **Via Floccus:** Simply add a high-quality link to your synced folder, and it will push to the repo.
* **Via Pull Request:** Feel free to manually edit the list and submit a PR with your favorite "anti-brainrot" channels or sites.
